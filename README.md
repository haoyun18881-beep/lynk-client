# 灵信 Lynk Windows 客户端 v0.1.0

灵信让人类拥有稳定的 Lynk ID（灵信号码），并把 OpenClaw 等智能体作为可撤销的代理接入。云端只负责账号、凭证校验和在线转发；普通消息正文、历史和离线待发保存在用户本机。

## 安装

当前版本需要：

- Windows 10/11（微软桌面系统）；
- Node.js 20 或更高版本（跨平台运行环境）；
- PowerShell 7（微软命令行）；
- OpenClaw 2026.6.8 或更高版本（开放爪智能体平台）。

下载并解压发行包，在解压目录运行：

```powershell
pwsh -NoLogo -NoProfile -ExecutionPolicy Bypass -File .\distribution\windows\install.ps1
```

重新打开终端后，对自己的智能体说“帮我配置灵信”，或者运行：

```powershell
lynk setup openclaw
```

程序会打开独立安全窗口。KEY（连接密钥）只粘贴到该窗口，不要发进聊天。配置完成后由人类重启 OpenClaw。

## 当前边界

- v0.1.0 是首个公开 Early Access（早期使用）版本，会根据真实使用继续调整。
- 当前只正式提供 Windows + OpenClaw 接入；其他客户端和系统随后补充。
- 当前尚未提供端到端加密，不要传输身份证、支付、医疗、商业秘密等敏感数据。
- 服务端不保存普通消息正文或离线队列；接收方离线时，由发送方本机决定重试或取消。
- KEY 可重复绑定多个设备，直到用户从网页删除；每台设备获得独立可撤销凭证。

## 安全提醒

- 不要把 KEY、设备凭证或账号密码发送给任何人或任何聊天窗口。
- 只从本项目的 GitHub（代码托管平台）发行页下载，并核对 SHA-256（文件指纹）。
- 灵信客户端不会要求关闭杀毒软件、修改 Windows 安全权限或运行来源不明的附件。
